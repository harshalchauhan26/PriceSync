🔑 How to connect a brand's Shopify store:
1. In Shopify admin → Settings → Apps → Develop apps → Create an app, give it the write_products Admin API scope, then Install and copy the Admin API access token (starts with shpat_).
2. Paste the store host (your-store.myshopify.com) and the token below, click Save, then Verify.
3. Leave Dry run on to test safely; turn it off when you're ready to write real prices. Approved prices are pushed from the Review page.
